# synit.io Nexthink

Welcome to `synit-nexthink`, a repository of powerful PowerShell modules and scripts to enrich the Nexthink data model with external security and IT context.

## About Nexthink
Nexthink is a Digital Employee Experience (DEX) platform used by IT teams to monitor endpoint health, understand user impact, and drive proactive remediation. By enriching Nexthink with trusted third-party signals, teams can improve visibility, prioritization, and automated decision-making.

## Purpose of This Repository
This repository provides integrations and enrichment modules designed to:
- Correlate external telemetry with endpoint/application context
- Extend Nexthink data model attributes with actionable security insights
- Support scalable PowerShell-based data workflows for enterprise environments

## Available Modules
- [crowdstrike](./crowdstrike): CrowdStrike enrichment module for vulnerability correlation by file hash (`CrowdStrikeSoftwareVulnByHash.psm1`).
